# Essential Git Commands

* **git init**: Initialize a new local repository.
* **git clone**: Copy a remote repository to your machine.
* **git status**: Check the status of your files.
* **git add**: Stage changes for the next commit.
* **git commit**: Save staged changes to history.
* **git push**: Upload local changes to GitHub.
* **git pull**: Download and merge remote changes.
* **git branch**: Manage or list branches.
* **git checkout**: Switch between different branches.
* **git log**: View the commit history.
